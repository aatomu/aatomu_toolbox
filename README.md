# aatomu_toolbox
自分用のChrome拡張機能

## How to Install
1. [aatomu_toolbox](https://github.com/atomu21263/aatomu_toolbox/archive/refs/heads/main.zip)をDLする
2. DLしたファイルを展開する
3. [拡張機能](chrome://extensions/) を開く
4. デベロッパーモードをONにする
5. パッケージ化されていない拡張機能を読み込むをクリックし展開した`aatomu_toolbox-main`を選択する
6. 以上!
